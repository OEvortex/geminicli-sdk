# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial multi-language SDK release
- Python SDK with full feature support
- TypeScript SDK with full feature support
- Rust SDK with full feature support
- Go SDK with full feature support
- C++ SDK with full feature support

### Features (All Languages)
- OAuth authentication using Gemini CLI credentials
- Automatic token refresh
- Streaming responses (SSE)
- Non-streaming responses
- Tool/function calling
- Thinking/reasoning mode
- Session management with event system
- Comprehensive error handling

## [0.1.0] - 2026-01-26

### Added
- Initial release
- Python SDK
- TypeScript SDK
- Rust SDK
- Go SDK
- C++ SDK
- GitHub Actions CI/CD workflows
- Automatic version syncing across all SDKs
- Auto-publish to PyPI, npm, crates.io on release
